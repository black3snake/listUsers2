# ListUsers2

Angular version 16.
используется бек написанный на JS (Node.js).

1. Отображает список пользователей (массив объектов с полями)
   - id,firstName,lastName,age,experience,address,phone,email
2. Добавлено поле поиска, которое фильтрует список по имени (fullName).
3. Список для фильтрации: показать всех, только активных или только неактивных пользователей.
4. При клике на пользователя выводит его данные в отдельном блоке.
5. Возможность редактировать данные пользователей.
6. Создавать и удалять пользователей.
7. Загружать аватарку для пользователя.
